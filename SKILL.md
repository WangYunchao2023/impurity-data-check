# impurity-data-check

HPLC有关物质检测数据明显错误检查工具。

## 报告限配置

**阈值优先级（从高到低）：**

| 优先级 | 来源 | 说明 |
|--------|------|------|
| 1 | `--threshold` 命令行参数 | 全局单一阈值 |
| 2 | `impurity-checker.yaml` 配置文件 | 支持按杂质名称设置不同阈值 |
| 3 | Excel 表头自动识别 | 扫描"报告限：X%"等关键词 |
| 4 | 内置默认值 | 0.05% |

## 配置文件

放置于 **skill 目录**：`~/.openclaw/workspace/skills/impurity-data-check/impurity-checker.yaml`

```yaml
# ── 默认报告限（未匹配到杂质专用阈值时使用） ──
default_threshold: 0.05

# ── 按杂质名称设置不同报告限 ──
# 匹配规则：精确匹配 > 包含匹配
impurity_thresholds:
  # 已知杂质（精确名称匹配）
  "杂质M": 0.05
  "杂质N": 0.10

  # 其他单个杂质（包含匹配：杂质名包含此 key 即匹配）
  "其他单个杂质": 0.05

  # 示例：
  # "甲硝唑杂质A": 0.10
  # "甲硝唑杂质B": 0.05
```

**匹配规则：**
1. **精确匹配**：杂质名称 == 配置 key
2. **包含匹配**：杂质名称包含配置 key
3. 都匹配不到 → 使用 `default_threshold`

## 批号格式规范

标准格式：**项目编号-两位年两位月两位日-两位序号**

```
LMS002-260401-05
```

示例：
- `LMS002-260401-05` ✅ 合规
- `LMS002-26040105` ❌ 缺少连字符
- `LMS002-260401-5` ❌ 序号位数不足

## 检查项目

### 1. 批号格式一致性检查

- 检测同一文件内批号格式是否统一
- 识别缺少连字符的批号

### 2. 忽略不计判断合理性检查

- 标记"忽略不计"的含量值必须小于对应杂质的报告限
- 含量 ≥ 该杂质的报告限却标"忽略不计" → 报错

## 使用方法

```bash
python3 scripts/checker.py <文件路径> [--threshold <阈值>]
```

**示例：**

```bash
# 使用配置文件中的阈值
python3 scripts/checker.py LMS002-HPLC-有关物质.xlsx

# 临时指定全局阈值（覆盖配置文件）
python3 scripts/checker.py LMS002-HPLC-有关物质.xlsx --threshold 0.06
```
